Project 01 · SKIL 2.1

Hospital Navigation System

A group project focused on designing an internal navigation system for hospitals, with attention to accessibility, BLE technology and cybersecurity.

Open project PDF

Context

Large hospitals can be difficult to navigate. Patients and visitors can lose time, feel stress and arrive late. Our idea was to create a digital navigation system that helps users find the correct room or department more easily.

What we created

We designed a concept for a mobile navigation application using BLE beacons for indoor positioning. The system included route preferences, accessibility options, route calculation and a backend/database concept.

My contribution

I contributed to the analysis, concept thinking and documentation. I was also responsible for implementing the map logic where the user position (the moving indicator) is displayed and updated in real time. This included ensuring that the map dynamically refreshed based on the user's movement. During this project, I learned how important clear communication and structured task division are when working in a team.

What I learned

  • How to analyse a real-world problem.
  • How BLE can be used for indoor positioning.
  • How security frameworks help structure risks.
  • How to work better in a team environment.
