Welkom op mijn portfolio

Welkom op mijn portfolio. Deze website heb ik gemaakt voor het vak Skills Integration binnen mijn richting Cloud & Cybersecurity. Hier verzamel en toon ik mijn projecten, opdrachten en vaardigheden die ik tijdens mijn opleiding ontwikkel.

Op deze website kun je een overzicht vinden van mijn werk, zoals technische projecten, beveiligingsoplossingen en cloud-gerelateerde toepassingen. Het doel van dit portfolio is om mijn groei, kennis en praktische ervaring duidelijk te demonstreren.

Doorheen mijn opleiding werk ik aan verschillende cases en uitdagingen, en via deze website wil ik laten zien hoe ik problemen analyseer en oplos binnen de wereld van IT en cybersecurity.
